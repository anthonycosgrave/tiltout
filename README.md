tiltout
=======

HTML5 mobile clone of Breakout using DeviceOrientation Events for control.

I'm still learning JavaScript so be kind! 

Tested in Safari and Chrome on the following devices:

  iOS6
---------
iPhone 4,
iPhone 4S,
iPhone 5,
iPad 4,
and iPad Mini.

  Android
-----------
Samsung Galaxy S3,
and HTC One
